Calculadora de Integrales MAC - versión web

Abre codigo html/index.html en el navegador.

Incluye:
- Menú principal con integrales indefinidas, definidas, métodos, cinemática, tablas, ejemplos y glosario.
- Renderizado LaTeX con MathJax.
- Escritura de números como enteros, decimales o fracciones.
- Pasos en LaTeX para las operaciones de la calculadora.
- Carpeta backend-python con el proyecto Python original usado como referencia.

Nota: MathJax se carga desde CDN, por lo que el renderizado LaTeX completo requiere internet al abrir la página.


ACTUALIZACIÓN DEL GLOSARIO:
- Para ver los resultados del glosario, abre codigo html/diccionario.html desde el menú principal, opción 8.
- El glosario ahora permite buscar por palabra clave y filtrar por tipo.
- Las fórmulas del glosario se renderizan en LaTeX con MathJax.
- También se incluye codigo html/glosario.tex y codigo html/glosario.pdf compilado en LaTeX.

ACTUALIZACIÓN MÉTODOS DE INTEGRACIÓN:
- La página de métodos ahora usa backend Python con FastAPI.
- El usuario ya no debe escribir derivadas.
- Cambio de variable pide integrando y cambio g(x); Python calcula g'(x).
- Por partes pide solo el integrando; Python identifica f(x), g'(x), f'(x) y g(x), y aplica el teorema formal.
- Fracciones parciales acepta funciones racionales originales o ya separadas.

Para activar estos métodos:
1. Abrir terminal en backend-python/CalculadoraDeIntegrales
2. Ejecutar: pip install -r requirements.txt
3. Ejecutar: uvicorn app:app --reload
4. Abrir codigo html/metodos-integracion.html
