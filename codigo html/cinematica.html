CINEMÁTICA

Esta sección conecta cálculo integral con movimiento. Escribe una función de velocidad \(v(t)\), coloca el tiempo inicial y el tiempo final, y la página interpreta el movimiento con una gráfica. Está pensada como recurso físico-matemático para explicar qué significa acumular una velocidad.

Apoyo rápido antes de calcular

Velocidad \(v(t)\): indica qué tan rápido cambia la posición de un objeto en cada instante.

Desplazamiento: es el cambio neto de posición. Se obtiene acumulando la velocidad con signo; si la velocidad es negativa, resta.

Distancia recorrida: mide todo el camino recorrido. Para eso se acumula la rapidez \(|v(t)|\), así nada se cancela.

Gráfica: la curva muestra la velocidad y el sombreado representa acumulación durante el intervalo elegido.

Muestra desplazamiento, distancia recorrida, velocidad media y rapidez media usando la misma función.
